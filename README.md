# 🚀 Software Obfuscation Analyzer

An interactive web application that allows users to upload Python files or process existing files to analyze execution performance.

## 🔹 Features
- Process multiple Python files
- Simulate code obfuscation
- Track execution time
- Identify success/failure cases
- Interactive dashboard (Streamlit)

## 🔹 Tech Stack
- Python
- Pandas
- Streamlit

## 🔹 How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run backend (optional):
python main.py

3. Run app:
streamlit run app.py

## 🔹 Project Structure
- app.py → Dashboard
- main.py → Batch processing
- files/ → Sample files
- data.csv → Output data
